នេះជាការពន្យល់អំពី output នៃកម្មវិធីរបស់អ្នក! 😊

🔐 ផ្នែកទី 1 — Login (ចូលប្រើប្រាស់)
  1. Login
  2. Register new account
  3. Exit

Login = ចូលប្រើប្រាស់គណនីដែលមានស្រាប់
Register = បង្កើតគណនីថ្មី
Exit =ចេញពីកម្មវិធី


📋 ផ្នែកទី 2 — Main Menu (ម៉ឺនុយចម្បង)
  1. Add Transaction
  2. View All Transactions
  3. Monthly Summary
  4. Delete Transaction
  5. Search by Category
  6. Save & Logout

Add Transaction = បញ្ចូលប្រាក់ចូល ឬប្រាក់ចេញថ្មី
View All Transactions = មើលប្រតិបត្តិការទាំងអស់
Monthly Summary = សង្ខេបប្រាក់ចូល-ចេញតាមខែ
Delete Transaction = លុបប្រតិបត្តិការ
Search by Category = ស្វែងរកតាមប្រភេទ (ម្ហូប, ថ្លៃជួល...)
Save & Logout = រក្សាទុក ហើយចេញ


➕ ផ្នែកទី 3 — Add Transaction (បញ្ចូលប្រតិបត្តិការ)
  Type:
    1. Income   ← ប្រាក់ចូល (ប្រាក់ខែ, おbonusជាដើម)
    2. Expense  ← ប្រាក់ចេញ (ម្ហូប, ថ្លៃជួលជាដើម)

  Category: salary      ← ប្រភេទ
  Amount: 1500          ← ចំនួនទឹកប្រាក់
  Date: 2026-03-09      ← កាលបរិច្ឆេទ
  Note: monthly salary  ← កំណត់ចំណាំ

📊 ផ្នែកទី 4 — View Transactions (មើលប្រតិបត្តិការ)
  ID   Type      Category   Amount     Date         Note
  ----------------------------------------------------------------
  1    income    salary     $1500.00   2026-03-09   monthly salary
  2    expense   food       $50.00     2026-03-09   lunch

ID = លេខរៀងប្រតិបត្តិការ
Type = ប្រភេទ (income=ចូល / expense=ចេញ)
Category = ប្រភេទចំណាយ
Amount = ចំនួនទឹកប្រាក់
Date = កាលបរិច្ឆេទ
Note = កំណត់ចំណាំ


📅 ផ្នែកទី 5 — Monthly Summary (សង្ខេបប្រចាំខែ)
  Month     Income       Expense      Balance
  -----------------------------------------------
  2026-03   $1500.00     $50.00       $1450.00  [OK]

Income = ប្រាក់ចូលសរុប
Expense = ប្រាក់ចេញសរុប
Balance = សល់ប្រាក់
[OK] = ល្អ មានប្រាក់សល់ 😊
[Over budget!] = ប្រើប្រាស់ច្រើនពេក ⚠️
